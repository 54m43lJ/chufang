# FAQ
- **项目怎么部署？**
- 参考[SETUP.md](SETUP.md)
- **为什么用腾讯云COS而不是微信原生的云存储？**
- 因为这是一个passion project，不是一个拿出去卖钱的商业项目，为了实现上传图片的功能支付微信云开发20元/月的订阅我实在是想不出理由。这不是一坨没事找事的屎山代码，而是慎重考虑过成本和可用性之后作出的工程决定。作为个人项目部署，发布体验版小程序是零成本，而腾讯云COS按用量付费成本可以以毛为单位，简直不要太爽。
- **那我钱多的慌想给腾讯送钱怎么办？**
- 图片管理的API已经在utils/api.js定义好了，后端逻辑在云函数image API里，你可以自己覆盖。
